# Остров Шкота media

Файлы подключены в `src/constants/images.ts` как `TOUR_SPRING_11_*`.

Фото:

- `hero.webp` — обложка карточки и hero страницы.
- `view.webp` — фон блока «О туре».
- `view2.webp`, `view3.webp`, `view4.webp`, `view5.webp`, `view6.webp`, `wow.webp` — видовые кадры.
- `climb.webp` — верёвочный участок / спуск.
- `scallops.webp`, `earchin.webp` — морепродукты.

Видео:

- `shk.clip1.webm`, `shk.clip2.webm`, `shk.clip3.webm` — viewer-версии.
- `shk.clip1.grid.webm`, `shk.clip2.grid.webm`, `shk.clip3.grid.webm` — сетка галереи.
- `shk.clip1.poster.webp`, `shk.clip2.poster.webp`, `shk.clip3.poster.webp` — постеры для ленивой загрузки видео.

Пересборка видео:

```powershell
.\scripts\encode-spring-11-shkota-movs.ps1
```
