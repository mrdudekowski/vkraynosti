# Мыс Тобизина media

`cover.webp` подключён в `src/constants/images.ts` как `TOUR_SPRING_12_COVER_GRID`.

Текущие файлы:

- `cover.webp` — обложка карточки и hero страницы (`content/HERO/Весна/tobizina.webp`).

Ожидаемые финальные файлы галереи:

- `preface.webp` — фон блока «О туре».
- `trail.webp` — тропа через лес к мысу.
- `stone-plateau.webp` — каменная «шахматная доска».
- `cliffs.webp` — скалы и обрывы мыса.
- `karpinsky-bay.webp` — бухта Карпинского / купание.
- `seafood-lunch.webp` — обед с морепродуктами на берегу.

После добавления остальных файлов замените placeholder-константы галереи `TOUR_SPRING_12_*` в `src/constants/images.ts` на пути из `TOUR_SPRING_12_ASSET_BASE`.
