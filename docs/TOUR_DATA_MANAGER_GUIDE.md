# Как обновлять туры и расписание на сайте

## Обновить только туры

**Вкрайности → Обновить туры**

## Обновить только расписание

**Вкрайности → Обновить расписание**

## Обновить всё

**Вкрайности → Обновить всё** (сначала туры, затем расписание)

## Черновик тура

Не ставьте статус в колонке **F** («Статус на сайте»).

## Черновик выезда

Не ставьте статус в колонке **H** («Статус»).

## «В разработке»

Карточка на сайте есть, выезды в календарь не попадают.

## «Скрыт»

Тур и выезды не показываются на сайте.

## Проблемы

Лист **«_Проверка»** в таблице.
